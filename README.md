# shorterid-cli

Leismore ShorterID Generator (CLI) is a command-line tool for generating new Leismore ShorterIDs.

## Installation

`npm install -g @leismore/shorterid-cli`

## Usage

`shorterid`, or

`npx @leismore/shorterid-cli`
